O trabalho Teatro utiliza fundamentos de Java para a criação de um servidor que suporte um sistema para reserva de lugares em um teatro.

Este sistema foi feito utilizando Java Servlets.

Tem como intuito a compreensão de filas, processos e threads.

Sistema criado para o trabalho de Sistemas Operacionais "A" no 4* semestre de 2023 do curso de Sistemas para Internet-UFSM.
